# 📊 Bank Loan Analysis Dashboard (Power BI)

## 🚀 Project Overview
This project presents an interactive Power BI dashboard designed to analyze bank loan performance, risk metrics, and trends.

---

## 📌 Key Features
- Total Loan Applications, Funded Amount, Amount Received
- Average Interest Rate & Debt-to-Income (DTI)
- Good vs Bad Loan Analysis
- Month-to-Date (MTD) & Month-over-Month (MoM) metrics
- Multi-page dashboard (Summary, Overview, Details)
- Dynamic filters (State, Grade, Purpose)
